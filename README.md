# Unlimited Handmade GP ITI

ITI Graduation Project - E-commerce platform for handmade products

## Structure

- `backend/` - Backend server (ASP.NET Core)
- `frontend/` - Frontend application (Angular)

## How to Run

### Backend
```bash
cd backend
dotnet run
```

### Frontend
```bash
cd frontend
npm install
npm start
```

## Features
- Product and shop management
- Authentication and role system
- Shopping cart
- Admin dashboard
